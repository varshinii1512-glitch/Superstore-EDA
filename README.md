# Superstore Sales Data Analysis (EDA)

## Project Objective
Perform Exploratory Data Analysis (EDA) on Superstore dataset
to understand sales, profit, and business performance.

## Dataset Information
- Total Rows: 9994
- Total Columns: 21
- No Missing Values
- No Duplicate Rows

## Key Business Insights
- Total Sales: 2297200.86
- Total Profit: 286397.02
- Sales & Profit Correlation: 0.47 (Moderate Positive)
- Business is overall profitable

## Tools Used
- Python
- Pandas

## Analysis Performed
- Data inspection (shape, columns, data types)
- Summary statistics
- Missing value check
- Duplicate check
- Correlation analysis
- Region-wise Sales & Profit
